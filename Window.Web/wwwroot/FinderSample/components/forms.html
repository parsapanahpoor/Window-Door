مشاده در سایت بوت استرپ

Range اسلایدر وابسته به noUISlider است. میبایست برای استفاده از پلاگین فایل های css و js noUISlider به در فایل پروژه خود تعریف نمایید بصورت : vendor/nouislider/distribute/nouislider.min.css و vendor/nouislider/distribute/nouislider.min.js

برای فعال کردن فرمت دهی محتوای متن ورودی ، لطفاً در فایل پروژه افزونه Cleave.js را لینک دهید بصورت : vendor/cleave.js/dist/cleave.min.js
